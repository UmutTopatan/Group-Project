## Prerequisites
* C++17 Compiler (GCC or Clang)
* CMake 3.10+
* PostgreSQL (libpqxx)

## Build Steps
1. Clone the repo: `git clone https://github.com/FurkanKayaFbu/Group-Project`
2. Create build folder: `mkdir build && cd build`
3. Configure: `cmake ..`
4. Compile: `make`
